# Basel IRB Credit Risk Capital Model

This project implements the **Basel II Internal Ratings-Based (IRB) model** to calculate regulatory capital for credit risk using Python.

The model applies the **Vasicek single-factor framework** to estimate capital requirements based on key credit risk parameters.

## Key Features

* Basel IRB capital calculation
* Expected Loss vs Regulatory Capital comparison
* Portfolio simulation of loans
* Sensitivity analysis for PD and asset correlation

## Tools

Python, NumPy, Pandas, SciPy, Matplotlib
